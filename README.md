# 品保部排程專案

這是一個給品保部使用的排程展示與流程管理 Demo，現階段以本機模擬與 GitHub 展示為主，後續可再部署到公司伺服器。

## 目前功能

- 品保部共用行事曆
- 驗證組與品檢組名單展示
- 主管層與部門成員分區
- 每日出差 / 請假名單
- 驗證排程建立
- 產品驗證時程
- 設備時程排程
- 超時 / 逾期判斷
- Discord 提醒模擬

## 重要檔案

- `demo-preview.html`
  單機展示版，直接雙擊即可預覽
- `open-demo.cmd`
  Windows 一鍵開啟展示頁
- `src/server.js`
  本機 API 與伺服器版本入口
- `public/`
  網頁版資源
- `data/app-data.json`
  假資料與展示資料

## 如何預覽

### 單機展示版

直接打開：

```text
demo-preview.html
```

或雙擊：

```text
open-demo.cmd
```

### 本機伺服器版

```bash
node src/server.js
```

再開啟：

```text
http://localhost:3000/
```

## 驗證排程欄位

建立排程時可填寫：

- 排程類型
- 啟動日期
- 預計時間
- 結束時間
- 提前提醒分鐘

系統會顯示：

- 產品驗證時程 / 設備時程排程
- 預估完成時間
- 是否超時
- 是否逾期

## 部門行事曆權限

- 品保部眾人可共同維護
- 品保經理可查看兩組全局時程
- 副理與組長可協助調整驗證與設備工作
